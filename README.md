# ESP32-Vibe-RGB

基于 ESP32 的高性能、响应式音频视觉渲染引擎。通过 I2S 数字麦克风采集音频，利用硬件 FFT（快速傅里叶变换）分析频域数据，并将其实时转化为 20+ 种绚丽的 RGB 矩阵特效。

---

## 🚀 核心特性

- **双核并发处理**：Core 1 负责 16kHz 高频音频采样与 DSP 分析，Core 0 负责 30FPS 丝滑特效渲染。
- **硬核音频算法**：
  - **FFT 分析**：256 点实时变换，将声音划分为 8 个频带。
  - **动态增益 (AGC)**：自动适应环境音量大小，确保视觉效果始终饱满。
  - **节拍检测 (Beat Detect)**：精准捕捉音乐节奏，触发闪烁与转场。
- **丰富的视觉库**：内置 22 种特效（包括彩虹、频谱、矩阵雨、等离子等），支持 2D 模糊滤镜和物理坐标映射。
- **Web 实时交互**：
  - **虚拟控制面板**：手机连接后可在线调节亮度、速度、色彩和特效切换。
  - **WebSocket 同步**：网页端实时预览 LED 矩阵画面。
  - **持久化存储**：所有设置自动保存至 ESP32 NVS 闪存。

## 🛠️ 硬件清单

| 组件 | 推荐型号 | 备注 |
| :--- | :--- | :--- |
| **主控** | ESP32 / ESP32-S3 | 需支持双核以保证渲染流畅度 |
| **麦克风** | INMP441 / ICS-43434 | I2S 数字麦克风（音频质量远超模拟麦克风） |
| **显示屏** | WS2812B RGB 矩阵 | 默认支持 8x8 (64 颗灯珠)，可扩展 |
| **电源** | 5V 2A 以上 | 视 LED 数量而定 |

### 默认引脚配置 (GPIO)

| 功能 | 引脚 | 备注 |
| :--- | :--- | :--- |
| **LED 数据** | GPIO 16 | 连接 WS2812 DIN |
| **I2S SCK** | GPIO 4 | 麦克风时钟 |
| **I2S WS** | GPIO 5 | 麦克风声道选择 |
| **I2S SD** | GPIO 6 | 麦克风数据输入 |

## 📥 编译与烧录

本项目使用 **ESP-IDF v5.0+** 开发。

1. **配置环境**：
   ```bash
   . $HOME/export.sh
   ```
2. **构建项目**：
   ```bash
   idf.py build
   ```
3. **烧录并监控**：
   ```bash
   idf.py flash monitor
   ```

## 📱 使用指南

1. **配网**：首次启动，ESP32 会进入 **WiFi Provisioning** 模式，可通过网页或热点配置家中 WiFi。
2. **访问控制台**：设备联网后，在浏览器中输入设备 IP（从串口日志获取）。
3. **实时调节**：
   - 切换 **FX** 菜单选择不同音效模式。
   - 调节 **Sensitivity** 改变声音灵敏度。
   - 调节 **Palette** 改变整体色调。

---

## 📂 项目结构

- `main/mic.c`: 音频采集、AGC、FFT 及节拍算法实现。
- `main/effects.c`: 22 种视觉特效逻辑。
- `main/led.c`: WS2812B 驱动、矩阵映射及 2D 渲染引擎。
- `main/wifi_ctrl.c`: HTTP Server 与 WebSocket 实时通讯协议。
